# Personal Portfolio

A portfolio website built with [Dioxus](https://dioxuslabs.com/).

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js](https://nodejs.org/) and npm
- Dioxus CLI: `cargo install dioxus-cli`

## Run

```bash
npm install
dx serve
```

## Build

```bash
dx build --release
```
